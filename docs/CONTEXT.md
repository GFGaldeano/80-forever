# CONTEXT.md

## Visión general

**80's Forever** es una plataforma web de streaming musical ochentoso diseñada como una señal digital temática con identidad propia.

No es una web decorativa ni una simple página con video embebido.

Busca combinar:

- experiencia audiovisual
- branding fuerte
- monetización visual con sponsors
- operación simple desde admin
- crecimiento posterior hacia contenido y comunidad

---

## Idea del producto

La propuesta parte de un concepto televisivo/musical adaptado al presente:

- transmisión central de música de los 80
- bloques especiales por artista o banda
- sponsors visibles durante la experiencia
- gestión manual y simple desde un dashboard
- posibilidad de evolución hacia pedidos musicales, blog y programación

---

## Modelo operativo

La arquitectura operativa del producto es:

- **OBS Studio** produce la señal
- **YouTube Live** distribuye la transmisión principal
- la **web presenta** el player y el contexto visual
- el **panel admin** permite controlar lo que se ve

## Objetivo del MVP

La primera versión debe resolver:

- actualización del stream desde admin

- base técnica seria y escalable
